鼠标右键菜单选项

在处理 GML 可视化代码时，您可以在操作、工作区或左侧短名单中的文本上单击鼠标右键  ，打开鼠标右键菜单：

该菜单有以下选项：

  • Cut / Copy / Paste: 这样就可以剪切、复制或粘贴工作区中选中的任何操作。请注意，您可以在操作之外使用  +  并拖动，或在单个操作上使用  /  +  ，选择多个操作并将命令应用于所有操作。
  • Comment / Remove Comment: 该选项允许您添加和删除 GML 可视化代码块中的注释。选择 "注释" 后，选定的 GML 可视化代码块右侧将打开文本编辑器，您可以在其中添加任何所需的文本，无论是提醒您更改某些内容，还是为教程添加有用的注释，甚至是为您的队友提供指导：
    添加注释后，您可以单击并拖动它，使其在垂直和水平方向上定位（但只能在父节点的右侧），折叠 GML 可视化节点也会折叠注释，并显示其内容摘要（在这种情况下，注释无法重新定位）。如果不再需要注释，可以删除注释中的所有文本，或使用菜单中的 "删除注释 "选项。请注意，复制节点时，注释也会一起复制，如果将节点从 GML Visual 转换为 GML，那么注释将被保留并添加到创建的 GML 中，如下图所示：
    如上所述，注释只能应用于单个节点，但您可以通过添加换行符来扩展注释，使其覆盖多个节点，不过这可能需要重新定位注释的位置，以防止与后续节点重叠：
    添加评论后，您可以再次从 RMB 菜单中选择 "Comment" 选项，也可以双击  进行编辑。
  • Disable / Enable: 这些命令可用于禁用或启用所选动作。当一个动作被禁用时，它在工作区中仍然可见，但在编译游戏时不会运行。通过这种方式，您可以添加/删除动作，用于测试和调试。下图显示了一些禁用动作的 GML 可视化代码：
    您可以看到 "If Score（如果得分）"和 "Go To Next Room（转到下一个房间）"这两个操作已被禁用--从视觉上看，这两个操作是灰色的，因此在测试或编译游戏时，这两个操作将被跳过。
  • Toggle Breakpoint: 该命令将打开或关闭所选动作的断点。断点只是 GML Visual 代码中的一个点，当游戏在调试模式下运行时，调试器应在此停止。指定了断点的动作会突出显示，如下图所示（"If Health（如果健康）" 动作已被设置为断点）
  • Delete All Breakpoints: 使用该命令，您可以删除 GML 可视化代码中的所有断点。
  • Collapse All / Expand All: GML 可视化操作可以折叠或展开，折叠操作只显示操作名称，不显示任何内部值，从而最大限度地利用工作区的可用空间。例如：
    正如你所看到的，这些操作占用的空间要小得多，而且你仍然可以从右侧的简短说明中看到它们的作用。您可以选择 "全部展开 "选项来撤销工作区中每个节点的操作。
  • Reset Layout: 这将重置 GML 可视化工作区，将缩放级别设置为 100%，聚焦于当前选定的操作，并展开任何（和所有）折叠操作。
  • Live Preview: 选择此选项后，您将启用或禁用 GML 可视化转 GML（GameMaker 语言）代码预览窗口。转换 GML 可视化到代码页面对此有更详细的解释。
  • Convert To GML: 这将把当前事件中的所有操作转换为 GML（GameMaker 语言）代码。使用时要非常小心，因为您无法将 GML 转换回由它创建的 GML Visual 操作。您可以将事件从代码转换回 GML 可视化事件，但任何代码都将被简单地包裹在执行代码动作中。转换 GML 可视化到代码 页面对此有更详细的解释。
  • Window: 本节将打开以下子菜单：
    这里提供的菜单项目有：
    • Go To Object: 这将把工作区平移到调用事件的对象上
    • Open Inherited Event: 只有当被编辑对象是父对象的子对象时，该选项才可用，它将打开父对象的被编辑事件，并将工作区平移到该事件。
    • Single Pane / 2 Columns / 3 Columns: 这些选项可用于将当前工作区分割成 1、2 或 3 个不同的 "窗格"。每个窗格都是一个独立的工作区，可以将脚本或事件拖入其中，这样就可以同时编辑和比较 2 个或多个 GML Visual 代码块。下图显示了一个窗口中的两个脚本，显示在各自的工作区窗格中：

以下两个选项只有在 GML 可视化节点上打开 RMB 菜单时才可用：

  • Pan: 在任何节点上选择该选项时，工作区都将平移到该节点的中心位置。
  • Show Help...: 选择此选项将在与所选节点相关页面的新工作区中打开手册。您也可以通过在节点上按  来达到同样的效果。
