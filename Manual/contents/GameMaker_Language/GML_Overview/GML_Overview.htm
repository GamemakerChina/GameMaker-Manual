GML Code 概述

本手册的这一部分包含所有理解和使用 GML 代码所需的信息。这种语言的 Runtime 函数可以用来创建你的游戏，这些函数可以从 物体编辑器 添加到物体中，同时也可以与GML Visual结合使用。下面你可以看到一个典型的示例，展示了通过物体中的事件打开代码编辑器的样子：

每个事件在代码编辑器中都有自己的选项卡，你可以随时添加、编辑或删除其中的代码（有关事件相关的更多信息请查看 物体事件 ）。代码本身必须有基本结构，其中可能会包含资源索引、变量、函数、表达式或关键字等。所有的这些内容在接下来会逐一讲到。如果你是一个编程新手，或是准备从 GML Visual 转到 GML Code 的人，建议你从 基础代码结构 页面开始，紧接着你可以阅读本页面中的其他章节的页面。同时，你可以在 GameMaker 中测试每一部分的代码。

你可以使用 Feather 功能在GML代码中帮助你进行智能代码补全和错误报告。
